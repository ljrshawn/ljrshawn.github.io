# Personal Website

This is Shawn Lyu's personal resume and portfolio website, built with Next.js, Tailwind CSS, and shadcn-style UI components. It is statically exported for GitHub Pages.

Welcome to my personal web [shawnl.dev](https://shawnl.dev/).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

The static site is exported to `out/`.

## Deployment

GitHub Pages is deployed by `.github/workflows/deploy.yml`. In the repository settings, set Pages source to **GitHub Actions**.
